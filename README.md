I am learning **2D game design and interactive programming** using Python and Pygame.

> **Project 2.0** — a hand-tracking therapy game for paralyzed and disabled children, where kids do their rehabilitation exercises by playing a game using real hand movements detected by a camera.

These mini games are the foundation for that.
